# nepub-reader

小説家になろうの作品を縦書き表示で快適に読めるリーダーです。

## 特徴

- 縦書き表示で小説家になろうの作品を読める
- ダークモード対応（システム設定に連動）
- 矢印キー（←/→）で次/前のエピソードへ自動移動

## セットアップ

### 必要なもの

- Python 3.10 以上
- uv（Python パッケージマネージャー）

### インストール

```bash
uv sync
```

### サーバー起動

```bash
uv run nepub-reader
```

ブラウザで http://localhost:5000/ にアクセスしてください。

## 使い方

### 小説 ID で開く

トップページのフォームに小説 ID とエピソード番号を入力して「読む」ボタンを押します。

### URLで開く

小説家になろうの URL を直接入力することもできます。

- エピソードURL（`https://ncode.syosetu.com/nXXXXXX/1/`）→ その話から開始
- 小説トップURL（`https://ncode.syosetu.com/nXXXXXX/`）→ 1話から開始

### 小説 ID の見つけ方

小説家になろうの URL から取得できます：

```
https://ncode.syosetu.com/nXXXXXX/
                         ^^^^^^^ これが小説ID
```

### 操作方法

- **← / →キー**: 次/前のページへ移動（端に到達すると次/前のエピソードへ）
- **メニューバー**: 文字サイズ変更

## キャッシュ

生成された EPUB ファイルは `bibi-bookshelf/narou/` ディレクトリにキャッシュされます。
キャッシュを削除したい場合は、このディレクトリ内のファイルを削除してください。
